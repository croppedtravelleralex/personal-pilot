    什么时候用

    当你要把一个新的已授权站点接入系统，并让它可发布、可验证、可复盘。

    什么时候不用

    如果只是看系统是否正常、看最近任务和异常，不需要进入这一页。

    最少步骤

    登录地址 → 自动发现 → 补齐 password / submit / ready selector → 发布并验证。
