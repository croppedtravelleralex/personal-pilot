# FEATURES.md

`PersonaPilot` / `PersonaPilot` 项目最终目标功能文档。

目的：单独回答一个问题——

> 这个项目最终要实现哪些能力？

---

## 1. 产品目标

本项目最终要成为一个：

- 运行在 Ubuntu 上
- 面向个人学习与研发
- 具备高性能任务执行能力
- 可持续演进
- 具备文档化工程纪律
- 可从 fake runner 平滑升级到真实浏览器执行器

的浏览器自动化系统。

---

## 2. 最终核心功能

### 2.1 任务生命周期管理
- 创建任务
- 查询任务
- 排队任务
- 执行任务
- 取消任务
- 重试任务
- 成功 / 失败 / 超时状态管理

### 2.2 执行引擎抽象
- fake runner 闭环能力
- real runner adapter
- `lightpanda-io/browser` 集成能力
- runner trait / interface 统一

### 2.3 数据持久化
- SQLite 任务存储
- run history 持久化
- logs 持久化
- artifact 元信息管理
- 后续迁移/升级能力预留

### 2.4 REST API
- 健康检查
- 创建任务
- 查询任务状态
- 查询运行历史
- 查询执行日志
- 取消任务
- 重试任务
- 状态与摘要查询
- API 鉴权

### 2.5 指纹与网络策略
- 浏览器指纹配置与注入
- 指纹模板 / 指纹策略
- 高级指纹下的性能开销控制
- 代理池管理
- 代理抓取、清洗、去重、验证
- 代理轮换与失败剔除
- 代理池自生长
- 地区感知代理匹配
- 所有访问强制走代理池
- 指纹与代理协同策略

### 2.6 资源与稳定性控制
- 并发控制
- 资源限制
- 磁盘使用监控
- artifact / log 保留、清理、归档
- 写放大控制
- 长期运行稳定性保护

### 2.7 观测与验证
- 结构化日志
- 基础错误分类
- 最小 smoke test
- 执行验证机制
- 阶段汇总与结果追踪

### 2.8 高价值增强能力
- 身份画像系统（Identity Profile）
- 指纹一致性校验器
- 代理质量评分系统
- 站点维度代理适配
- 行为层模拟
- 会话连续性系统
- 策略引擎
- 实验记录系统

---

## 3. 非目标 / 暂不追求

在当前及近期阶段，不把以下事项当成首要目标：

- 多租户平台化
- 面向公网的大规模开放服务
- 一开始就追求极重的分布式架构
- 在 fake runner 阶段提前实现全部高级策略系统

---

## 4. 阶段映射

### 当前已具备（按方向）
- 最小任务闭环
- SQLite 持久化
- 内存任务队列
- fake runner
- runs / logs 查询接口
- 可选 API Key 鉴权
- Lightpanda runner 占位适配层

### 当前未完成但已进入主线
- 更完整的 runner adapter 演进
- 真实 Lightpanda 执行接入
- running cancel
- 查询控制 / 分页能力
- 更强控制面与观测面

### 中长期能力
- 指纹系统
- 代理池系统
- 身份画像与一致性系统
- 行为模拟
- 策略引擎
- 实验记录系统
